## 🔍 Overview
This project demonstrates a gesture-controlled robot using computer vision and embedded systems.  
Hand gestures are detected using a YOLOv8 pose model, and commands are sent wirelessly to an ESP32 to control robot movement.

---

## ⚙️ Tech Stack
- Python  
- OpenCV  
- YOLOv8 (Ultralytics)  
- ESP32  
- Arduino IDE  

---

## 🚀 How It Works
1. Camera captures the hand gesture  
2. YOLOv8 detects 21 hand keypoints  
3. Gesture is classified (Forward, Left, Right, Stop)  
4. Command is sent via WiFi  
5. ESP32 controls the motors
