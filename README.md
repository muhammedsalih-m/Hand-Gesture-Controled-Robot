# Gesture Controlled Robot 🤖✋

A robot controlled using real-time hand gestures with computer vision and ESP32.

## 🔍 Overview
This system detects hand gestures using a YOLOv8 pose model and sends commands wirelessly to an ESP32.

## ⚙️ Tech Stack
- Python
- OpenCV
- YOLOv8 (Ultralytics)
- ESP32
- Arduino IDE

## 🚀 How it Works
1. Camera captures hand gesture  
2. YOLO detects 21 keypoints  
3. Gesture classified (Forward, Left, Right, Stop)  
4. Command sent via WiFi  
5. Robot moves  

## 📊 Results
- Real-time performance  
- High accuracy in controlled environment
